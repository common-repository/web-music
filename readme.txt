== Description ==

Music Web is a plugin for Wordpress that lets you insert music players or recordings in your blog post. This makes it easy to add music to the post in your blog, and podcast for personal recordings.
Platform that supports:

* Goear
* Ijigg
* Evoca
* MyBloop
* Deezer
* Spool.fm

How to use:

Put the url between the tags [mus] and [/ mus]

[mus]http://www.goear.com/listen/9676ce6/Boulevard-of-Broken-Dreams-Green-Day[/mus] <br />
[mus]http://www.ijigg.com/songs/V2ABEE4PB0[/mus] <br />
[mus]http://www.evoca.com/everyone_recording.jsp?rid=11154[/mus] <br />
[mus]http://mybloop.com/go/J91e16[/mus] <br />
[mus]http://www.deezer.com/track/1277819[/mus] <br />
[mus]http://spool.fm/#track/LTCr2ZWy/Britney+Spears/Gimme+More[/mus] <br />

== Installation ==

1. Upload web-music directory in the wp-content/plugins directory
2. Activate the plugin
3. Add music and recordings to your post!
